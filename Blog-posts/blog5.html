Wat doet het bedrijf?

Oracle Corporation is een leverancier van software en cloudtechnologie. De techreus vergaarde bekendheid met zijn Oracle databasesysteem. Naast cloudtechnologie en databases is Oracle Corporation ook een van de grootste leveranciers van bedrijfssoftware verzameld onder de naam Oracle E-Business Suite (ERP-/CRM-pakket).

Verschillende ICT-rollen

  • Software Development Engineer
  • Product/Program Manager
  • Leadership
  • Network/Data Center
  • National Security Region

Verschillende Technologiën

  • Oracle Database
  • MySQL
  • MS SQL Server
  • PostgreSQL

Bedrijfscultuur

Iedereen wordt gewaardeerd. Onze verschillen maken ons sterk. Wij geloven dat verschillende gezichtspunten essentieel zijn voor innovatie. Bij Oracle maak je deel uit van een inclusieve cultuur waar individualiteit gedijt. Er is dus een vrije sfeer, waar iedereen op een collegiale manier met elkaar omgaat.

Conclusie

Als ik naar de werksfeer van dit bedrijf kijk, valt me op dat alles er heel ontspannen aan toe lijkt te gaan. Ik vind het ook mooi om te zien, dat ze van zo'n groot belang zijn voor zoveel bedrijven.
